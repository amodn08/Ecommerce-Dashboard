# Ecommerce-Dashboard
This project is an end-to-end Excel-based analytics dashboard built on top of a real-world style e-commerce dataset. It showcases data cleaning, transformation, KPI derivation, pivot modeling, and dashboard design â€” perfect for demonstrating Excel + analytics skills.

## Project Overview

The dataset contains:
25,000+ transactions, 
Order & shipping dates, 
Product categories, 
Customer gender,
Order modes (App, Website, Phone), 
State & county level data, 
Ratings and delivery duration

The dashboard provides a clear business snapshot for decision-makers.

## KPIs Included
Total Sales & Quantity Sold, 
Average Rating, 
Orders by Gender, 
Orders by State / Region, 
Top-performing product categories, 
Delivery Time (Days to Deliver), 
Weekly order trends

## What the Dashboard Shows
Interactive filters, 
Clean, color-coded visuals, 
Pivot charts for sales, product mix, customer segments, and regional performance, 
Insightful trend lines for weekly behavior

## Skills Demonstrated
Power Query, 
Pivot Tables & Pivot Charts, 
DAX-like logical KPIs within Excel, 
Data cleaning & transformation, 
End-to-end KPI dashboard design, 
Storytelling through visuals
